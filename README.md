# SmartCanteen — система рекомендаций блюд

## Описание проекта

SmartCanteen — это веб-приложение для формирования персональных рекомендаций блюд в системе столовой.  
Сервис анализирует историю заказов пользователей и с помощью модели машинного обучения предлагает блюда, которые пользователь с наибольшей вероятностью выберет.

Система состоит из следующих компонентов:

- **Frontend** — пользовательский интерфейс (React + Vite)
- **Backend** — REST API сервис (FastAPI)
- **ML-модель** — модель рекомендаций CatBoost
- **База данных** — PostgreSQL

Модель машинного обучения анализирует характеристики пользователя, блюда и контекст заказа (день недели, время суток) и формирует список рекомендованных блюд.

---

## Установка и запуск

Проект разворачивается с использованием **Docker и Docker Compose**.

### Требования

Перед запуском необходимо установить:

- Docker  
- Docker Desktop  
- Docker Compose  

### Запуск системы

Перейдите в папку проекта:

```bash
cd SmartCanteen
```

Запустите систему:

```bash
docker compose up --build
```

Docker автоматически запустит следующие сервисы:

- backend (FastAPI)
- frontend (React)
- database (PostgreSQL)

---

## Доступ к системе

После запуска сервис будет доступен по следующим адресам:

| Сервис | Адрес |
|------|------|
| Backend API | http://localhost:8000 |
| API документация | http://localhost:8000/docs |
| Frontend интерфейс | http://localhost:5173 |

## Примеры использования

### Получение списка блюд через API

Запрос:

```
GET http://localhost:8000/dishes
```

Пример ответа:

```json
[
  {
    "id": 1,
    "name": "Куриное филе",
    "category": "main",
    "price": 200
  }
]
```

---

### Получение рекомендаций

Запрос:

```
GET http://localhost:8000/recommendations/{user_id}
```

Пример:

```
GET http://localhost:8000/recommendations/1
```

Сервис вернёт список рекомендованных блюд для пользователя.

---

### Использование через веб-интерфейс

Откройте в браузере:

```
http://localhost:5173
```

Через интерфейс можно:

- просматривать меню блюд
- оформлять заказы
- получать персональные рекомендации
